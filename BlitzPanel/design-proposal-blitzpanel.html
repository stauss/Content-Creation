00 Project Overview

Blitzpanel has a clear product, real customers, and a compelling mission. What it needs now is a design foundation that communicates speed, clarity, and engineering confidence — a visual identity and landing experience that converts the right audience from "never heard of it" to "I need to try this."

This is a focused, fast engagement: brand direction, a slim Framer design system, a full landing page at desktop and mobile, and a clean developer handoff — all completed in under three weeks. Every phase has a clear approval gate so there are no surprises.

Fixed fee. No hourly surprises. Scope changes beyond two revision rounds are quoted in writing before any additional work begins.

01 Phases & Deliverables
01 Discovery & Strategy 4 hrs $340

Structured intake questionnaire completed async before a focused kickoff call. We align on audience, goals, content requirements, and competitive positioning — and produce a one-page project brief both sides sign off on before any design work begins.

Intake questionnaire Kickoff call (60 min) Competitive brand audit Signed project brief
02 Moodboarding & Brand Direction 7 hrs $595

Blitzpanel doesn't yet have a defined visual identity. Two distinct mood board directions are presented — different visual worlds, type pairings, and color approaches. One is selected and refined into an approved Brand Direction document that governs all subsequent design work.

2 mood board directions Typography pairing options Color palette proposals Photography / visual tone direction Presentation call + feedback
03 Slim Design System 9 hrs $765

Before the landing page, we build the system it runs on — a lean but complete Framer component library: design tokens, type scale, core UI components, and spacing grid. Built so the client can construct future pages independently without starting from scratch.

Color + spacing tokens Type scale (5 levels) Button variants Card & badge components Nav + footer components Framer library + naming conventions
04 Landing Page Design 20 hrs $1,700

The core deliverable. Full high-fidelity landing page at desktop and mobile — all sections, all states. Includes a content architecture document and section-by-section copy scaffold to direct what gets written where. Delivered as a clickable Framer prototype the client can share with stakeholders and their dev team.

Content architecture doc Copy scaffold Desktop hi-fi (all sections) Mobile hi-fi Interaction & animation direction notes Clickable Framer prototype Breakpoint specs
05 Revisions — 2 Rounds 6 hrs $510

Two rounds of structured revisions with a short alignment call before changes are applied — no guessing at written notes. Both rounds cover desktop and mobile. A third round is available as a paid add-on.

Round 1 feedback call Round 1 revisions Round 2 feedback call Final approved designs
06 Handoff & Dev Documentation 6 hrs $510

Annotated designs with spacing, type, color, and interaction specs — everything the developer needs to implement without follow-up questions. Plus launch-ready meta assets and a handoff walkthrough call.

Annotated designs + markup Component + spacing specs Animation behavior notes Asset export package OG image (1200×630) Favicon set Social card template Handoff walkthrough call
07 Project Management & Communications 2 hrs $170

Shared Notion board, milestone tracking, and all async comms throughout the engagement. Because the project runs under 3 weeks, check-ins are milestone-driven rather than scheduled weekly calls.

Shared Notion project board Milestone tracking Async comms
